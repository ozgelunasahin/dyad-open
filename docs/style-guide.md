# UI Style Guide

## Text Casing

### Headings (Title Case)
Page titles, section headers, and modal titles use Title Case - capitalize major words.

**Examples:**
- My Conversations
- Discover
- Start a Conversation
- Meetings
- Getting Started

### Buttons & Actions (lowercase)
All interactive elements like buttons, links that act as buttons, and action labels use lowercase.

**Examples:**
- start a conversation
- sign out
- delete
- cancel
- send invitation
- sending...

### Badges & Labels (Sentence case)
Status indicators and informational badges use sentence case.

**Examples:**
- Published
- Read-only
- Draft

## Rationale

This style creates a calm, modern aesthetic that:
- Reduces visual noise by avoiding ALL CAPS
- Feels approachable with lowercase actions
- Maintains readability with sentence case for headings
- Creates clear hierarchy between headings and actions
