# Show engagement status on discover cards

## Problem Statement

When Tom responds to Sophie's conversation, it still appears on discover as a regular card. Tom has no visual cue that he's already engaged with this conversation. A subtle indicator ("You responded" or "Meeting scheduled") would help Tom track which conversations he's involved in without navigating to his profile.

## Source

- User testing discussion: conversations you're involved in should be recognisable on discover
